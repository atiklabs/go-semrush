# Go-SEMRush
A small client for SEMRush API.
## Usage
```
Usage of go-semrush:
   -api string
     	API Key given by SEMRush
   -f string
     	CSV file with the url in the first column
   -h	First row of file is header (default true)
   -lang string
     	SEMRush language database (default "us")
```
